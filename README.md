# 📝 To-Do List App

تطبيق بسيط باستخدام JavaScript لإدارة المهام (To-Do List)، حيث يمكنك إضافة المهام وحذفها بسهولة.

---

## 🚀 فكرة المشروع

التطبيق يسمح لك بـ:

- كتابة مهمة في حقل الإدخال
- الضغط على زر "إضافة"
- ظهور المهمة في القائمة
- حذف أي مهمة عن طريق زر "حذف"

## 🎯 ماذا تعلمت من المشروع؟

- التعامل مع `input.value`
- إنشاء عناصر جديدة باستخدام `createElement`
- إضافة عناصر للصفحة باستخدام `appendChild`
- حذف العناصر باستخدام `remove`
- التعامل مع الأحداث `onclick`
- فهم أساسيات DOM Manipulation

## ▶️ طريقة تشغيل المشروع

1. قم بتحميل المشروع.
2. افتح ملف `index.html` في المتصفح.
3. اكتب مهمة واضغط "إضافة".
4. يمكنك حذف أي مهمة.
